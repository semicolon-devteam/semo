### Migration Guide

**모든 SEMO 사용자**:

1. `/SEMO:update` 실행 시 CLAUDE.md가 자동으로 최신 템플릿으로 갱신됩니다
2. 기존 프로젝트는 업데이트 후 Orchestrator-First Policy가 자동 적용됩니다

**신규 프로젝트 배포 시**:

1. `package-deploy` 스킬이 자동으로 CLAUDE.md 템플릿을 설치합니다
2. 별도 설정 없이 Orchestrator-First Policy가 적용됩니다
