# 0.56.0 (2024-12-30)

## Added

### meta-workflow 스킬 추가

Meta 환경에서 semo-system/ 수정 시 자동으로 실행되는 기본 워크플로우 스킬:

- **기본 활성화**: Meta 패키지 설치 환경에서 semo-system/ 수정 감지 시 자동 실행
- **명시적 트리거**: `[Meta]`, `[meta]`, `메타 작업` 프리픽스
- **워크플로우 체인**: 환경 검증 → 작업 수행 → 버저닝 & 배포 → 로컬 동기화

### 워크플로우 4단계

| Phase | 동작 |
|-------|------|
| Phase 1 | 환경 검증 (Meta 패키지, GitHub 인증, 브랜치) |
| Phase 2 | 작업 수행 (semo-system/ 수정) |
| Phase 3 | 버저닝 & 배포 (VERSION 범프, CHANGELOG, git push, Slack) |
| Phase 4 | 로컬 동기화 (심볼릭 링크 재생성) |
