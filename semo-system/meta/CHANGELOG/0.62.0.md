# meta v0.62.0

> Released: 2025-02-03

## Added

### write-portfolio Skill
- 프로젝트 포트폴리오 문서 자동 생성 스킬 추가
- 레퍼런스 링크 (URL, Notion, GitHub) 분석 기능
- 소스코드에서 기술스택/기능 자동 추출
- Playwright MCP를 활용한 서비스 스크린샷 자동 캡처
- `docs/portfolio/{project-name}/` 구조로 문서 생성
  - README.md, PROPOSAL.md, TECH_STACK.md 템플릿
  - images/, documents/ 디렉토리 구조

### 주요 기능
- GitHub MCP로 저장소 분석 (package.json, build.gradle 등)
- WebFetch로 서비스/문서 URL 콘텐츠 분석
- 반응형 스크린샷 캡처 지원 (모바일 뷰포트)
