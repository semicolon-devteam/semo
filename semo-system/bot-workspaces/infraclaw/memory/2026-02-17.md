### 컨테이너 오케스트레이션: OKE (Oracle Kubernetes Engine)

- **변경 이력**: 초기 ARM A1.Flex → AMD E4.Flex/E5.Flex로 전환
  - 이유: GitHub Actions에서 ARM 이미지 빌드가 유료 (QEMU 에뮬레이션 시 30분 이상 소요)
- **비용 비교**:
  - ARM (A1.Flex 2OCPU 8GB): ~$25/월
  - AMD E4.Flex (2OCPU 8GB): ~$47/월 (노드 ~$37 + LB ~$10)
- **ArgoCD**: 개발 환경 설정 완료 (정보 탭에서 확인 가능)
- **OKE 노드 증설**: OCI 측에 증설 신청, 답변 대기 중
