# Core Backend (Spring Boot) 아키텍처 컨텍스트

> 출처: #core-backend 채널 논의 (2026-02-14, Reus & Garden)

## 현황
- 현재 랜드 서비스들이 **core-backend** 단일 Spring Boot 서버에서 운용 중
- 백엔드 분리 필요성은 합의됨

## 분리 방향 (미확정)
- **도메인별 분리** vs **서비스별 분리** — 아직 결정 못한 상태
- Garden 초기 구상: **영역별 분리** (권한 / 포인트 / 게시글 등)
- 진행하다 어려움 발생 → "일단 하던 대로 하고 뜯자"로 방향 전환

## 마이크로서비스 전환 계획
- 채팅 모듈 같은 분리 가능한 것부터 천천히 마이크로서비스로 분리
- 구체적 일정/분기 미정 (이전에 논의한 적 있으나 기록 없음)
