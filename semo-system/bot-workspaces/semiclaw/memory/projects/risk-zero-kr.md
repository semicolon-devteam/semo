# 리스크제로 KR (국가철도공단 안전관리)

## 개요
- **유형**: 프리랜서 하청 (Reus)
- GH(경기건설) 건설→철도청 도메인 전환
- CCTV 대시보드+스트리밍+모바일
- 정부/공공기관 납품

## 레포 & 스택
- `kr-back` — Spring Boot 2.7 / Java 17
- `kr-front` — React 18 / Vite / MUI / Zustand
- `kr-mobile` — React 18 / 모바일웹
- `kr-cctv-dashboard` — Next.js 15 / HLS.js
- `kr-stream-server` — Spring Boot 3.5 / WebFlux / Elasticsearch / JavaCV
