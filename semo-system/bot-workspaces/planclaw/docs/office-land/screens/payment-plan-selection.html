🔄 사용자 플로우

1. 사용자 진입 (/enterprise/payment) ↓ 2. 3가지 플랜 확인 ↓ 3. 플랜 카드 클릭 → 선택 상태 변경 ↓ 4. [결제하기] 버튼 활성화 ↓ 5. 버튼 클릭 ↓ 6-A. Standard/Pro 선택 → Checkout 화면 이동 6-B. Enterprise 선택 → 문의하기 모달 오픈
