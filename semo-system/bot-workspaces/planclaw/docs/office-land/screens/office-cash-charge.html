🔌 API 명세

1. 잔액 조회 GET /rest/v1/points/balance?type=OFFICE_CASH Response: { "balance": 250000, "type": "OFFICE_CASH" } 2. 충전 요청 POST /rest/v1/points/office-cash/charge Body: { "amount": 100000, "paymentMethodId": "pm_xxx" } Response: { "success": true, "transactionId": "tx_xxx", "newBalance": 350000 } 3. 결제 검증 POST /rest/v1/payments/verify Body: { "transactionId": "tx_xxx", "paymentKey": "pk_xxx" } Response: { "status": "SUCCESS", "paidAt": "2026-03-04T10:00:00Z" }
