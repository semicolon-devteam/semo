### 3️⃣ **코인 상세 페이지 (`/coin/[symbol]`)** ⭐ 핵심!

예시: `/coin/BTC`

#### A. 헤더 섹션
- 코인 아이콘 + 이름 (Bitcoin BTC/USDT)
- 타임프레임 버튼 (15분, 1시간, 4시간, 일봉) - 선택 시 분석 업데이트
- 현재 가격 (`$68,013`)
- 24h 변화율
- 미니 차트
- 24h 고가/저가/거래량

#### B. AI 분석 엔진 (✦ AI ANALYSIS ENGINE)
- 현재 트렌드 ("약한 상승 흐름 (Weak Up)")
- 요약 메시지 ("상승 신호가 여러 개 보여요")
- 분석 기준 (1시간봉)
- 기준 가격대 / 상방 가격대 / 하방 가격대
- "분석 업데이트" 버튼
- "AI 상세 해석 (1시간봉)" 버튼 (클릭 시 LLM 기반 상세 해석 생성)
- AI 신뢰도 점수 (원형 게이지, 0-100)
- 면책 문구 ("매수/매도 신호가 아닌 상태 알림입니다")

#### C. 기술적 지표 섹션 (📊 기술적 지표)
12개 지표 카드 (각 지표별 상승/하락/중립 색상 표시):
1. **RSI(14)**: 값 + 해석 (과열/상승세/하락세 등)
2. **MACD**: 값 + 해석 (골든크로스/데드크로스)
3. **볼린저 밴드**: 위치 (상단/중단/하단)
4. **SMA(20)**: 현재 가격 위치 (위/아래) + 추세
5. **EMA(50)**: 위치 + 지지/저항
6. **스토캐스틱**: 값 + 과열/과매도
7. **CCI**: 값 + 해석
8. **ADX**: 값 + 추세 강도
9. **일목균형표**: 구름 위치 + 돌파 여부
10. **Williams %R**: 값 + 과열/과매도
11. **MFI**: 값 + 중립/과열
12. **ATR**: 변동성 수준

각 지표별:
- 지표명
- 현재 값 (숫자 또는 위치)
- 해석 텍스트
- 색상 인디케이터 (🟢 상승 / 🔴 하락 / 🟡 중립)

#### D. 파생상품 분석 섹션 (📈 파생상품 분석)
- 미결제 약정 (Open Interest) - 달러 금액 + 24h 변화율
- 롱/숏 비율 (바이낸스 상위 트레이더 기준)
  - % 막대 차트 (Longs / Shorts)
  - 해석 메시지 ("롱 포지션이 우세합니다. 펀딩비 상승 가능성에 주의하세요.")

#### E. 주요 지표 카드 섹션 (⚡ 주요 지표)
- **펀딩비** (Funding Rate): % + 연율 환산
- **24시간 거래량**: 달러 + 전일 대비 % 변화
- **시가총액**: 달러 + 도미넌스 %
- **NVT 비율**: 값 + 평가 (Overvalued/Undervalued)

#### F. 실시간 청산 섹션 (🔥 실시간 청산)
- "● LIVE" 인디케이터
- 1H 롱 청산 금액
- 1H 숏 청산 금액
- 최근 청산 내역 (최대 4건):
  - 시간 ("9분 전")
  - 포지션 (Long/Short)
  - 청산 가격 (`@$67,635`)
  - 청산 금액 (`$271.5K`)

#### G. 거래소 순유출입 (💰 거래소 순유출입)
- 순 유출입 (BTC 또는 달러)
- 상태 표시 ("균형" / "순유입" / "순유출")
- Inflow / Outflow 수치

#### H. 고래 활동 (🐋 고래 활동)
- "100만달러 이상 거래" 필터
- 최근 고래 거래 (지갑 주소 + 금액 + 시간)

## 📸 Playwright 캡처 작업 플로우

### 1단계: 페이지별 캡처
각 페이지를 Playwright로 열고 풀페이지 스크린샷 촬영:
- `/home`
- `/coin/BTC`, `/coin/ETH`, `/coin/SOL`
- `/chat`
- `/pricing`
- `/portfolio`
- `/profile` (개요, 구독, 사용량 분석, 설정 탭 각각)

### 2단계: 요소별 스냅샷
각 페이지에서 주요 컴포넌트를 개별 캡처:
- 헤더 (Ticker + Nav)
- 카드 (코인 신호 카드, ETF 카드 등)
- 섹션 (기술 지표, 파생상품 분석, 청산 등)
- 폼 (AI 채팅 입력, 로그인 폼 등)

### 3단계: 인터랙션 캡처
- 버튼 클릭 (타임프레임 전환, AI 해석 생성 등)
- 모달/드롭다운 열기
- 탭 전환 애니메이션
