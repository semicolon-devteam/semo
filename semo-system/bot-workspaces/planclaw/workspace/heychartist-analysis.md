## 📋 서비스 개요

**HeyChartist**는 암호화폐 차트 분석을 AI가 자동으로 해주는 서비스입니다. 초보 투자자들이 복잡한 기술적 지표를 이해하지 못해도, AI가 실시간으로 상승/하락/중립 신호를 분석하고 한글로 쉽게 설명해줍니다.

**핵심 가치제안:**
- 코인 이름만 입력하면 3초 안에 AI가 7개 지표를 분석
- 전문가 용어 없이 한글로 쉽게 설명
- 24/7 실시간 분석 (감정 없는 데이터 기반 분석)
- 가입 없이도 무료로 사용 가능

**타겟 고객:**
- 차트를 볼 줄 모르는 암호화폐 투자 초보자
- 감정적 투자보다 데이터 기반 의사결정을 원하는 투자자
- 매일 시장을 체크하는 활발한 트레이더 (Pro 플랜)

### 2. **코인 상세 분석 페이지** (/coin/BTC, /coin/ETH 등)

#### 2.1 **AI 분석 엔진 (핵심 기능)**
- **분석 기준 시간 프레임:** 15분/1시간/4시간/일봉 선택 가능
- **AI 판단:** 상승 흐름 / 하락 흐름 / 중립 / 약한 상승 등
- **신뢰도 점수:** 0~100점 (예: 66/100)
- **주요 가격대 표시:**
  - 기준 가격대 (현재 가격)
  - 상방 가격대 (저항선)
  - 하방 가격대 (지지선)

- **AI 상세 해석 버튼 (Pro 기능):**
  - 클릭 시 "왜 이런 신호가 나왔는지" 근거 설명
  - 예: "가격이 최근 평균(SMA20) 아래에 있어요 — 단기 약세 신호"
  - 주요 가격대 및 리스크 요소 상세 설명

#### 2.2 **기술적 지표 (12개)**
각 지표마다 현재 값 + 신호 (상승/하락/중립) + 간단한 설명
- RSI(14): 과열/침체/중립 판단
- MACD: 골든크로스 ↑ / 데드크로스 ↓
- 볼린저밴드: 상단/중간/하단 위치
- SMA(20): 가격 vs 평균선 위치
- EMA(50): 강한 지지/저항 여부
- 스토캐스틱, CCI, ADX, 일목균형, Will %R, MFI, ATR

#### 2.3 **파생상품 분석**
- 미체결약정(OI) 24시간 변화율
- 롱/숏 비율 (바이낸스 상위 트레이더)
- 롱 포지션 우세 시 경고 메시지

#### 2.4 **주요 지표 위젯**
- 펀딩비 (연율 환산)
- 24시간 거래량 (전일 대비 % 변화)
- 시가총액 + 도미넌스
- NVT 비율 (Overvalued/Fair 구간)

#### 2.5 **실시간 청산 모니터**
- 1시간 롱/숏 청산 규모
- 최근 청산 내역 (시간, 방향, 가격, 규모)

#### 2.6 **거래소 순유출입**
- Inflow / Outflow 비교
- 균형 상태 표시

#### 2.7 **고래 활동**
- 100만 달러 이상 거래 추적
- 대형 거래 건수 + 총 거래량

## 💡 특이사항 및 주목할 만한 기능

### 1. **데이터 근거 투명성**
- 분석 결과에 "왜?"를 제시 (AI 상세 해석 버튼)
- 7개 핵심 지표 + 시장 심리 + 고래 동향 + 청산 데이터 종합
- 매수/매도 신호가 아닌 "상태 알림"임을 명시 (투자 자문 회피)

### 2. **한글 친화성**
- 전문 용어 최소화
- 이모지 활용한 친근한 설명
- "골든크로스 ↑", "데드크로스 ↓" 같은 직관적 표현

### 3. **실시간성**
- 30초마다 새로운 분석 업데이트
- 청산 알림 LIVE 표시
- 시장 데이터 실시간 반영

### 4. **멀티 타임프레임 지원 (Pro 기능)**
- 단기(15분) ~ 장기(일봉) 시간대별 분석 가능
- 시간대에 따라 신호가 달라짐 (예: 1시간 "하락", 4시간 "약한 상승")

### 5. **AI 채팅의 자연스러움**
- GPT 기반으로 추정 (답변 스타일)
- 맥락 이해하고 추가 질문 제안
- "손절이나 목표가 기준도 알려드릴까요? 😊" 같은 친근한 대화

### 6. **법적 면책 강조**
- 모든 페이지 하단: "매수/매도 신호가 아닌 상태 알림"
- Footer: "본 사업자는 실시간 매매 신호를 제공하지 않습니다"
- "투자 자문이 아니며, 최종 투자 결정과 책임은 이용자에게 있습니다"
