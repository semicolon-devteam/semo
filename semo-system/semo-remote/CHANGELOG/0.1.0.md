# 0.1.0 (2025-12-26)

## Added

- **semo-remote 패키지 초기 릴리스**
  - Claude Code 원격 제어를 위한 Extension 패키지
  - 모바일 PWA에서 권한 요청 승인/거부, 사용자 입력 처리 가능

### 구성 요소

- **DB 스키마** (`db/schema.sql`)
  - `remote_requests` 테이블: 원격 요청 저장/관리
  - `remote_sessions` 테이블: 세션 상태 추적
  - RLS 정책 및 뷰/함수 포함

- **Hook 확장** (semo-hooks에 추가)
  - `PermissionRequest` Hook: 도구 실행 권한 원격 승인
  - `Notification` Hook: idle_prompt 감지 및 알림

- **MCP 도구** (semo-integrations에 추가)
  - `semo_remote_request`: 원격 요청 생성
  - `semo_remote_respond`: 원격 요청 응답
  - `semo_remote_pending`: 대기 중인 요청 조회

- **Orchestrator 및 Skill**
  - `remote-bridge` 스킬: 원격 상태 확인 및 관리

- **CLAUDE.md 프롬프트 규칙**
  - AskUserQuestion 호출 시 자동 MCP 도구 호출 규칙

## Architecture

```
┌─────────────────────────────────────────────────────────────────────┐
│                         semo-remote 패키지                           │
├─────────────────────────────────────────────────────────────────────┤
│  Layer 1: Hooks (semo-hooks 확장)                                   │
│  Layer 2: Prompt (CLAUDE.md 규칙)                                   │
│  Layer 3: DB (remote_requests, remote_sessions)                     │
│  Layer 4: Mobile PWA (향후 구현)                                    │
└─────────────────────────────────────────────────────────────────────┘
```
