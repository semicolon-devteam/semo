# 3.5.0

## Added

- **ideate 스킬**: 아이디어 탐색부터 Epic 생성까지 원스톱 워크플로우
  - Phase 1: Brainstorming (3단계 - 이해 → 탐색 → 합의)
  - Phase 2: 기술 검증 (spike 자동 연계)
  - Phase 3: Epic 생성 (dev-checklist 자동 검증)
  - Design Brief 산출물 자동 생성
  - References: brainstorming-guide, dev-checklist, design-brief-template

## Changed

- **create-epic**: ideate 스킬과 연계
  - Design Brief 입력 자동 수신
  - 호출 방식 테이블 추가 (ideate/epic-master/직접)
  - Related Skills에 ideate 추가

- **generate-spec**: Phase 0 분리
  - Brainstorming은 ideate 스킬로 이관
  - Phase 1-5만 담당 (specify → clarify → plan → checklist → tasks)
  - 러프한 아이디어는 ideate 스킬 사용 권장 안내

- **estimate-epic-timeline**: draft-task 참조 제거
  - tasks.md 기반 일정 예측으로 변경
  - Related Skills 업데이트

- **orchestrator**: ideate 라우팅 추가
  - "아이디어", "구상", "만들고 싶어" → skill:ideate
  - Available Skills에 ideate 추가
  - Available Skills 테이블 최신화 (#110)
    - tester → write-test, request-test 분리
    - coder → write-code 명칭 변경
    - verifier → create-impl-plan, trigger-deploy 분리

## New Workflow

```text
[러프 아이디어]
      ↓
ideate (Brainstorming + spike + Epic)
      ↓
generate-spec (specify → plan → tasks)
      ↓
create-issues (GitHub Issues)
      ↓
start-task → implement
```

## Design Brief Structure

```markdown
# Design Brief: {기능명}

## 1. 문제 정의
## 2. 목표 (Goals / Non-goals)
## 3. 사용자 시나리오
## 4. 제약사항
## 5. 개발자 체크리스트
## 6. 성공 지표
```
