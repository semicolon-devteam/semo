# semo-core v3.9.0

> Release Date: 2026-01-10

## Features

### start-task 스킬 Speckit 통합 (#124)

- **Speckit Progress 확인**: 작업 시작 전 이슈의 Speckit Progress 체크리스트 자동 확인
- **미완료 Phase 감지**: specify/plan/tasks Phase가 미완료된 경우 먼저 실행 제안
- **generate-spec 연계**: 미완료 Phase 발견 시 자동으로 generate-spec 스킬 호출 옵션 제공

### Speckit 스킬 위치 안내 (#123)

- semo-core/skills/에 존재하는 Speckit 관련 스킬 위치 문서화
- generate-spec, start-task 스킬 호출 방법 가이드
