## Migration Guide

### 기존 프로젝트

1. `.claude/agents` 심볼릭 링크 업데이트:
   ```bash
   rm -f .claude/agents
   ln -s ../semo-system/semo-core/agents .claude/agents
   ```

2. `.claude/skills` 심볼릭 링크 업데이트:
   ```bash
   rm -f .claude/skills
   ln -s ../semo-system/semo-core/skills .claude/skills
   ```

3. Runtime 설정 (선택):
   ```bash
   # .claude/memory/runtime.md 생성
   echo "Primary: nextjs" > .claude/memory/runtime.md
   ```

### 스킬 호출 방식

```text
# Before
[nextjs] 구현해줘
skill:implement (packages/eng/nextjs에서 로드)

# After
구현해줘
→ Runtime 자동 감지 → references/runtimes/nextjs 참조
→ skills/implement 또는 skills/nextjs-implement 호출
```
