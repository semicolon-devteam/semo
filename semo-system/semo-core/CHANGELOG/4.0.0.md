# semo-core v4.0.0

> Released: 2026-01-11

## BREAKING CHANGE

### skills 폴더 제거

`semo-core/skills/` 폴더가 `semo-skills/`로 통합되어 제거되었습니다.

**이전:**
```
semo-core/
├── skills/           ← 제거됨
├── agents/
├── commands/
└── principles/
```

**이후:**
```
semo-core/
├── agents/
├── commands/
└── principles/
```

## Removed

### 48개 스킬 → semo-skills로 이전

모든 Speckit 워크플로우 스킬이 `semo-skills/`로 통합되었습니다:
- create-epic, generate-spec, start-task, ideate
- deploy-service, health-check, debug-service
- analyze-code, run-code-review, validate-architecture
- 기타 38개 스킬

## Migration

스킬 참조 경로가 변경되었습니다:
- 이전: `semo-core/skills/{skill-name}`
- 이후: `semo-skills/{skill-name}`

Orchestrator 라우팅은 자동으로 `semo-skills/`를 참조합니다.
