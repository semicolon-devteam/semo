# 3.2.0 (2024-12-30)

## Changed

### 스킬 리네이밍 - 명시적 Action-Oriented 네이밍 적용

역할(Role) 지칭 스킬명을 구체적인 행동(Action) 중심 이름으로 변경:

| Before | After | 설명 |
|--------|-------|------|
| `verify` | `validate-pr-ready` | PR 제출 전 검증 |
| `review` | `run-code-review` | 코드 리뷰 실행 |
| `spike` | `explore-approach` | 기술 접근법 탐색/프로토타이핑 |
| `spec` | `generate-spec` | SDD 워크플로우 명세 생성 |
| `constitution` | `manage-principles` | 프로젝트 원칙 관리 |

### Orchestrator 라우팅 테이블 업데이트

- semo-skills 리네이밍 반영 (6개)
- semo-core 리네이밍 반영 (5개)
- 총 11개 스킬 라우팅 경로 갱신

## Notes

- 내부 구현 로직 변경 없음 (네이밍만 변경)
