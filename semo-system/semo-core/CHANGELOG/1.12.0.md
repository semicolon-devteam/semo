# 1.12.0 (2025-12-27)

## Added

- **동적 Onboarding 명령어 (`/SEMO:onboarding`)**
  - 설치된 패키지에 따라 맞춤형 온보딩 가이드 동적 생성
  - 패키지별 사용 가능한 기능/스킬 목록 표시
  - 권장 다음 단계 (추천 패키지) 제안
  - 패키지별 튜토리얼 스킬 연결 안내

## Changed

- `onboarding.md` 명령어 구조 개선
  - 정적 안내 → 동적 생성 방식으로 변경
  - 패키지 감지 로직 및 매핑 테이블 추가
  - 권장 조합 규칙 정의
