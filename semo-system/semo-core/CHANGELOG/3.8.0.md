# 3.8.0

## Changed

- **create-tasks → generate-spec 통합**: DDD Layer 분해 로직 중복 제거
  - create-tasks 스킬을 generate-spec에 병합
  - 단일 워크플로우로 통합: Epic → spec.md → plan.md → tasks.md → Task Issues
  - 트리거 통합: "태스크 만들어줘", "태스크카드 생성해줘" 등 모두 generate-spec으로 라우팅

- **ideate 출력 메시지 업데이트**
  - 다음 단계로 generate-spec 안내 (create-tasks 제거)
  - Related Skills 섹션 업데이트

## Removed

- `create-tasks` 스킬 제거 (generate-spec으로 통합)
  - 모든 기능은 generate-spec Phase 3-4에서 처리
  - layer-delegation.md 참조 문서는 generate-spec/references/로 이전

## 워크플로우 변경

**Before (v3.7.0)**:
```
ideate → Epic
           ↓
create-tasks → Task Issues
           ↓
generate-spec (Reverse Mode) → spec.md, plan.md
```

**After (v3.8.0)**:
```
ideate → Epic
           ↓
generate-spec → spec.md → plan.md → tasks.md → Task Issues
```
