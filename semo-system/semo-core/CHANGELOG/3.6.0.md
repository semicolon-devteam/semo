# 3.6.0

## Changed

- **ideate**: Design Brief 파일 대신 Epic Issue 직접 생성
  - Epic 본문에 Design Brief 내용 직접 포함 (Problem Statement, Goals, User Scenarios, Constraints, Success Metrics)
  - 개발자 체크리스트를 Task Issue로 위임 (Epic에서 제거)
  - 새 트리거 추가: "에픽 만들자", "에픽 만들어줘", "에픽 만들어", "epic 만들어", "Epic 생성해줘"

- **create-epic**: Epic 본문 템플릿 변경
  - Design Brief 링크 제거 (내용이 직접 포함되므로)
  - 개발자 체크리스트 섹션 제거 (Task로 위임)
  - 새 Epic 본문 구조: Problem Statement, Goals, User Scenarios, Constraints, Success Metrics

- **create-issues**: Task 본문에 개발자 체크리스트 섹션 추가
  - Epic에서 위임된 개발자 체크리스트를 각 Task에 포함
  - 데이터 흐름, 시간/계산, 플랫폼 제약, 도메인 지식 카테고리
