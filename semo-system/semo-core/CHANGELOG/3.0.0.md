**유지된 에이전트**:

## Improvements

### health-check 통합 스킬
- 플랫폼 자동 감지 로직 추가
- Runtime 기반 도구 검증 분기 (nextjs, spring, infra, design, generic)

### 아키텍처 단순화
- Orchestrator → Skill 직접 라우팅
- 중간 에이전트 레이어 제거
- 라우팅 복잡도 감소

## Migration Guide

### 스킬 참조 변경

| Before | After |
|--------|-------|
| `spring-implement` | `implement` (Runtime 자동 감지) |
| `nextjs-git-workflow` | `git-workflow` |
| `spring-health-check` | `health-check` (플랫폼 자동 감지) |

### 에이전트 참조 변경

| Before (Agent) | After (Skill) |
|----------------|---------------|
| `agent:implementation-master` | `skill:implement` |
| `agent:sprint-master` | `skill:project-board` |
| `agent:deploy-master` | `skill:deployer` |
