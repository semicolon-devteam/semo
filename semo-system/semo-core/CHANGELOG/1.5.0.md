# [1.5.0] - 2024-12-20

## Added

- `/SEMO:review` 커맨드 추가
  - 통합 리뷰 스킬 호출
  - 플랫폼별 리뷰 지원 (nextjs, spring, ms, infra)
  - PR에 리뷰 코멘트 자동 등록

- Orchestrator 라우팅 테이블에 `skill:review` 추가
  - 키워드: "리뷰", "/SEMO:review", "PR 리뷰"

## Changed

- GitHub Actions 기반 분산 리뷰 프로세스 → SEMO 통합 리뷰로 전환

## Migration

- 기존 GitHub Actions 리뷰 사용자는 [마이그레이션 가이드](../../docs/MIGRATE_REVIEW_TO_SEMO.md) 참조
