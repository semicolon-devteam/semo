# 1.15.0 (2024-12-30)

## Changed

### 스킬 리네이밍 - 명시적 Action-Oriented 네이밍 적용

역할(Role) 지칭 스킬명을 구체적인 행동(Action) 중심 이름으로 변경:

| Before | After | 설명 |
|--------|-------|------|
| `tester` | `write-test` | 테스트 코드 작성 및 실행 |
| `planner` | `create-impl-plan` | 구현 계획 생성 |
| `deployer` | `trigger-deploy` | GitHub Actions 배포 트리거 |
| `implement` | `write-code` | 코드 작성/수정 |
| `memory` | `persist-context` | 세션 컨텍스트 저장 |
| `feedback` | `create-feedback-issue` | 피드백 GitHub 이슈 생성 |

## Notes

- 내부 구현 로직 변경 없음 (네이밍만 변경)
- Orchestrator 라우팅 테이블 동기화 완료 (semo-core 3.2.0)
