# 1.10.0 (2025-12-28)

## Changed

- **tester 스킬 역할 세분화**
  - 테스트 코드 작성/실행/커버리지 확인만 담당
  - QA 테스터 할당 → `change-to-testing` (ops/qa)로 분리
  - 프로젝트 보드 상태 변경 → `change-to-testing` (ops/qa)로 분리
  - Slack 테스트 알림 → `request-test` (biz/management)로 분리
  - 역할 범위 테이블 추가

- **planner 스킬 역할 세분화**
  - 기능 구현/리팩토링 계획만 담당
  - Sprint 계획 → `sprint-master` (biz/management)로 분리
  - Roadmap 계획 → `roadmap-planner` (biz/management)로 분리
  - 계획서 템플릿 추가 (기능 구현, 리팩토링)
  - 역할 범위 테이블 추가

## Added

- **테스트 유형별 작성 가이드** (tester)
  - Unit Test 템플릿
  - Integration Test 템플릿
  - E2E Test (Playwright) 템플릿

- **계획서 템플릿** (planner)
  - 기능 구현 계획 템플릿
  - 리팩토링 계획 템플릿
