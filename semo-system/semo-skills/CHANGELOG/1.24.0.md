## Added

### create-meeting-minutes 스킬 추가

정기 회의록을 GitHub Discussion에 자동 생성하는 스킬입니다.

**주요 기능:**
- 이터레이션 기반 제목 자동 생성 (`{year}-{month}-{분자}/{분모}`)
- command-center 레포 Meeting-Minutes 카테고리에 Discussion 생성
- ISO Week 기준 월별 주차 계산
- macOS/Linux 호환 날짜 처리

**사용 예시:**
```bash
/create-meeting-minutes          # 현재 날짜 기준
/create-meeting-minutes 2026-02-15  # 특정 날짜 지정
```

**제목 형식:**
```text
2026-01-2/5  # 1월 2주차, 1월은 5주
2026-02-3/4  # 2월 3주차, 2월은 4주
```
