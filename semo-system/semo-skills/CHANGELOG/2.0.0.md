# semo-skills v2.0.0

> Released: 2026-01-11

## BREAKING CHANGE

### semo-core/skills 통합

`semo-core/skills/`의 모든 스킬이 `semo-skills/`로 통합되었습니다.

**이전 구조:**
```
semo-core/skills/     (48개 스킬 - Speckit 워크플로우)
semo-skills/          (22개 스킬 - 범용)
```

**새 구조:**
```
semo-skills/          (68개 스킬 - 통합)
```
