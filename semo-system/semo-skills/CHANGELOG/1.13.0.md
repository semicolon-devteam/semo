# 1.13.0 (2025-12-28)

## Changed

### 상세 스킬 버전 통합
- semo-core의 상세 버전 스킬을 semo-skills로 병합
  - circuit-breaker: 무한 루프 방지 상세 로직 추가
  - feedback: 이슈 템플릿 및 트리거 감지 상세화
  - list-bugs: 필터링 옵션 추가
  - memory: 메모리 스키마 및 동기화 로직 추가
  - notify-slack: 채널 설정 및 메시지 템플릿 상세화
  - semo-architecture-checker: 수정 로직 추가
  - semo-help: 문서 통합 및 팀 컨텍스트 추가
  - version-updater: 업데이트 프로세스 상세화

## Fixed

### 스킬 일관성 개선
- 모든 기본 스킬이 상세 문서화된 버전으로 통합
- references/ 디렉토리 추가로 Progressive Disclosure 적용
