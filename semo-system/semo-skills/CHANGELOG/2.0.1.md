## Fixed

### notify-slack 채널 ID 하드코딩 제거

채널 ID를 하드코딩하지 않고 Slack API를 통해 동적으로 조회하도록 변경했습니다.

**변경 전:**
```bash
"channel": "C09KNL91QBZ"  # 하드코딩된 ID
```

**변경 후:**
```bash
CHANNEL_NAME="_협업"
CHANNEL_ID=$(curl ... | python3 -c "...")  # API 동적 조회
"channel": "$CHANNEL_ID"
```

### 주요 변경사항

- `SKILL.md`: Step 2에 채널 ID 동적 조회 로직 추가
- `references/channel-config.md`: 하드코딩된 토큰/ID 제거, 동적 조회 예시로 대체
- NON-NEGOTIABLE 규칙 추가: "채널 ID를 하드코딩하지 마세요"
