# meeting Skill

> 미팅 안건, 요약, 회의록 관리

## Actions

| Action | 설명 | 트리거 |
|--------|------|--------|
| **agenda** | 안건 추가 | "안건 추가", "미팅 주제" |
| **summarize** | 미팅 요약 | "미팅 요약" |
| **minutes** | 회의록 작성 | "회의록 작성" |

---

## Action: agenda (안건 추가)

### 안건 템플릿

```markdown
# 미팅 안건 - {날짜}

## 1. {주제}
- **발표자**: {이름}
- **시간**: {분}
- **내용**: {요약}

## 2. {주제}
- **발표자**: {이름}
- **시간**: {분}
- **내용**: {요약}
```

---

## Action: summarize (미팅 요약)

### 요약 템플릿

```markdown
# 미팅 요약 - {날짜}

## 주요 논의사항
1. {주제}: {결론}
2. {주제}: {결론}

## 결정사항
- [ ] {액션 아이템} - @{담당자}
- [ ] {액션 아이템} - @{담당자}

## 다음 미팅
- **날짜**: {날짜}
- **안건**: {주제}
```

---

## Action: minutes (회의록)

### 회의록 템플릿

```markdown
# 회의록 - {날짜}

## 참석자
- {이름1}
- {이름2}

## 안건
1. {주제}

## 논의 내용
- {논의1}
- {논의2}

## 결정사항
- [ ] {액션 아이템} - @{담당자} - {기한}

## 다음 미팅
- **날짜**: {날짜}
```

[SEMO] Skill: meeting 완료 (minutes)
