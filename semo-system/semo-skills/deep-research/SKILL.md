# deep-research Skill

> Gemini Deep Research / Deep Think 최적화 프롬프트 생성

## Purpose

사용자가 제공한 컨텍스트(PDF, 문서, 대화 내용 등)를 분석하여 Gemini Deep Research 또는 Deep Think에 적합한 구조화된 프롬프트를 생성합니다.

## Trigger Keywords

- "Deep Research 프롬프트 만들어줘"
- "딥리서치용 프롬프트"
- "Gemini 연구 프롬프트"
- "Deep Think 프롬프트"
- "연구 질문 생성해줘"

## Workflow

### 1. 컨텍스트 수집

사용자로부터 다음 정보를 수집:

| 입력 유형 | 예시 |
|----------|------|
| 문서/PDF | 투자 제안서, 사업계획서 |
| 대화 내용 | 미팅 노트, 브레인스토밍 |
| 웹 URL | 참고 기사, 경쟁사 정보 |
| 질문/목표 | "MVP 아이디어 검증" |

### 2. 컨텍스트 분석

```markdown
## 분석 항목

1. **핵심 주제 추출**: 문서의 주요 주제 3-5개
2. **이해관계자 파악**: 관련된 인물, 조직, 시장
3. **현재 상태 정리**: 문제점, 기회, 제약사항
4. **목표 명확화**: 사용자가 원하는 결과물
```

### 3. 프롬프트 구조화

**출력 포맷**:

```markdown
# Deep Research 프롬프트

## 배경 컨텍스트
{핵심 정보 요약 - 500자 이내}

## 연구 대상
- **주체**: {회사/개인/서비스}
- **도메인**: {산업/분야}
- **목적**: {연구 목표}

## 연구 질문 (Research Questions)

### 시장 분석
1. {시장 규모/트렌드 관련 질문}
2. {경쟁 환경 관련 질문}

### 기술 검증
3. {기술 가능성 관련 질문}
4. {구현 방안 관련 질문}

### 사업성 검토
5. {수익 모델 관련 질문}
6. {확장성 관련 질문}

## 검증 필요 데이터

| 데이터 유형 | 구체적 항목 |
|------------|------------|
| 시장 데이터 | {필요한 통계} |
| 사례 연구 | {벤치마크 대상} |
| 전문가 의견 | {인터뷰 대상} |

## 기대 결과물
- {연구 완료 후 예상되는 인사이트}
- {의사결정에 활용할 정보}

---
💡 이 프롬프트를 Gemini Deep Research에 붙여넣어 사용하세요.
```

## 프롬프트 유형별 템플릿

### 1. MVP/사업 아이디어 검증

```markdown
# Deep Research: {아이디어명} MVP 검증

## 배경
{아이디어 요약}

## 연구 질문
1. 유사 서비스의 시장 현황과 성공/실패 요인은?
2. 타겟 고객군의 실제 니즈와 페인포인트는?
3. 기술적 구현 가능성과 예상 개발 비용은?
4. 초기 시장 진입 전략으로 적합한 방법은?
5. 수익화 모델의 현실성은?

## 검증 데이터
- TAM/SAM/SOM 추정
- 경쟁사 분석 (최소 5개)
- 고객 세그먼트별 특성
```

### 2. 투자/파트너십 분석

```markdown
# Deep Research: {대상} 투자 분석

## 배경
{투자 제안 요약}

## 연구 질문
1. 해당 산업의 성장 전망과 리스크는?
2. 경쟁사 대비 차별화 포인트는?
3. 경영진의 역량과 트랙레코드는?
4. 재무 구조의 건전성은?
5. Exit 시나리오와 예상 수익률은?

## 검증 데이터
- 산업 리포트 (최근 2년)
- 유사 투자 사례 및 성과
- 규제 환경 변화
```

### 3. 기술 트렌드 분석

```markdown
# Deep Research: {기술명} 트렌드 분석

## 배경
{기술 도입 검토 배경}

## 연구 질문
1. 현재 기술 성숙도와 향후 발전 방향은?
2. 주요 플레이어와 생태계 구조는?
3. 도입 시 예상되는 비용과 ROI는?
4. 리스크 요인과 대응 방안은?
5. 우리 조직에 적합한 도입 시기는?

## 검증 데이터
- 가트너/포레스터 리포트
- 도입 사례 (성공/실패)
- 기술 벤더 비교
```

## Output Example

**입력**:
> 홍명희 대표에게 피칭할 투자 제안서 + MVP 아이디어 브레인스토밍 요청.
> 대상: 의류 중고 판매 + 작업복 소매업 사업자

**출력**:

```markdown
# Deep Research 프롬프트

## 배경 컨텍스트
의류 중고 판매와 작업복 소매업을 영위하는 사업자를 대상으로 한 MVP 아이디어 발굴 및 투자 제안 준비. 기존 오프라인 소매업에 기술 솔루션을 접목하여 사업 확장 기회를 모색.

## 연구 대상
- **주체**: 홍명희 대표 (의류/작업복 소매업)
- **도메인**: 중고 의류, 작업복, 리테일 테크
- **목적**: MVP 아이디어 발굴 및 투자 유치

## 연구 질문 (Research Questions)

### 시장 분석
1. 국내 중고 의류/스포츠용품 시장의 규모와 성장률은?
2. AI 기반 가격 책정 솔루션의 기존 사례와 정확도는?

### 고객 분석
3. 외국인 관광객의 국내 소매점 이용 패턴은?
4. 다국어 서비스에 대한 니즈와 기대 효과는?

### 기술 검증
5. 중고품 상태 평가를 위한 이미지 인식 기술 현황은?
6. 소규모 매장에 적합한 POS/재고 관리 솔루션은?

### 사업성 검토
7. 중고 거래 플랫폼의 수수료 모델과 마진 구조는?
8. 작업복 B2B 시장에서 디지털화 기회는?

## 검증 필요 데이터

| 데이터 유형 | 구체적 항목 |
|------------|------------|
| 시장 규모 | 국내 중고 의류 시장 (2023-2025) |
| 고객 통계 | 외국인 관광객 쇼핑 지출 현황 |
| 경쟁사 | 번개장터, 당근마켓, 중고나라 분석 |
| 기술 사례 | AI 가격 책정 (ThredUp, Poshmark) |

## 기대 결과물
- 3개 이상의 검증된 MVP 아이디어
- 각 아이디어별 시장 기회 및 리스크 분석
- 투자 유치를 위한 데이터 기반 근거

---
💡 이 프롬프트를 Gemini Deep Research에 붙여넣어 사용하세요.
```
