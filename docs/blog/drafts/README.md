# 블로그 원고 디렉토리

> 이 디렉토리에는 `changelog-writer` 스킬이 자동 생성한 원고가 저장됩니다.

## 원고 생성 프로세스

```text
1. version-manager 완료
   ↓
2. changelog-writer 자동 호출
   ↓
3. 변경사항 분석
   ↓
4. 원고 생성 (drafts/)
   ↓
5. 수동 검토 및 편집
   ↓
6. docs/blog/로 이동
```

## 파일 명명 규칙

```
{YYYY-MM-DD}-{package}-{version}.md
```

예시:
- `2024-12-29-semo-core-2.2.0.md`
- `2024-12-29-semo-scripts-1.1.0.md`

## 원고 상태

| status | 설명 |
|--------|------|
| `draft` | 자동 생성된 초안 |
| `review` | 검토 중 |
| `ready` | 발행 준비 완료 |
| `published` | 발행됨 (docs/blog/로 이동) |

## 편집 가이드

1. **제목 다듬기**: 기술적 → 스토리텔링
2. **맥락 추가**: 왜 이 변경이 필요했는지
3. **예시 보강**: 실제 사용 예시 추가
4. **이미지/다이어그램**: 필요시 추가
5. **태그 정리**: 적절한 해시태그
