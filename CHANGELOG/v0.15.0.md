# v0.15.0

> Release Date: 2024-11-28

## Summary

Metabase typescript-write 패턴 적용: `_shared/` 디렉토리 도입으로 공용 리소스 중앙화 및 typescript-write/review skill 추가

## Added

### New Directory: `skills/_shared/`

공용 리소스 중앙화 (Metabase 패턴 적용):

- `development-workflow.md` - 개발 워크플로우 원칙 (TDD, 품질)
- `nextjs-commands.md` - Next.js 명령어 모음 (lint, test, build)
- `quality-gates.md` - 품질 기준 (ESLint, TypeScript, Coverage)
- `ddd-patterns.md` - DDD 4계층 패턴 및 규칙
- `ssr-rules.md` - SSR-First 규칙
- `test-templates.md` - Vitest 테스트 템플릿
- `browser-testing.md` - MCP 브라우저 테스트 가이드
- `commit-guide.md` - 커밋 규칙 및 워크플로우

### New Skills

- `typescript-write` - TypeScript/React 코드 작성 (DDD, TDD, Supabase 패턴)
- `typescript-review` - TypeScript/React 코드 리뷰 (품질, 아키텍처 검증)

## Changed

### @ 참조 문법 적용

기존 skills에 `@` 참조 추가로 공용 리소스 연결:

- `implement` - development-workflow, ddd-patterns, test-templates, commit-guide
- `verify` - quality-gates, ddd-patterns, browser-testing
- `validate-architecture` - ddd-patterns, ssr-rules
- `scaffold-domain` - ddd-patterns, test-templates
- `check-team-codex` - quality-gates, nextjs-commands
- `git-workflow` - commit-guide, quality-gates

### Documentation

- `README.md` - _shared 디렉토리 및 새 skills 문서화

## Benefits

| 항목 | Before | After |
|------|--------|-------|
| 중복 | references/ 분산 | _shared/ 중앙화 |
| 일관성 | skill별 상이 | 공용 원칙 통일 |
| 신규 skill | 전체 작성 | @참조로 간소화 |
| 역할 분리 | implement만 | write/review 분리 |

## Migration

기존 프로젝트는 변경 없이 호환. 새 skills 사용 시:

```javascript
// 코드 작성 시
skill: typescriptWrite();

// 코드 리뷰 시
skill: typescriptReview();
```
