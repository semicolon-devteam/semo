# 0.42.0

**Release Date**: 2025-12-04

## Added

### `spring-integration` Skill (신규)

Next.js 개발자를 위한 Spring Backend API 연동 가이드 스킬

**제공 기능**:
- cm-template API Client 패턴 가이드 (Factory + Interface)
- Spring ApiResponse → TypeScript 타입 매핑
- 에러 핸들링 패턴 (ApiError 클래스, React Query 연동)
- OpenAPI → TypeScript DTO 자동 생성 가이드
- core-backend/core-interface GitHub CLI 조회 명령어

**스킬 구조**:
```
skills/spring-integration/
├── SKILL.md
└── references/
    ├── api-client-pattern.md
    ├── api-response-mapping.md
    ├── error-handling.md
    ├── dto-generation.md
    └── github-commands.md
```

**사용 시점**:
- Spring API Client 구현 시
- ApiResponse/Error 매핑 필요 시
- DTO 타입 생성 필요 시
- 새 도메인 API Client 추가 시
