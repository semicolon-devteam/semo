# 0.30.0 (2025-12-01)

## Added

### Advisor Agent - SDD/Fast-track 선택지 우선 제시

- **작업 방식 선택지 우선 제시 (NON-NEGOTIABLE)**: 기능 구현 관련 질문 시 구체적인 구현 방안보다 SDD/Fast-track 선택지를 먼저 제시
- **트리거 패턴 추가**: `~하려면 어떻게`, `~하고 싶어`, `~구현하고 싶어`, `작업을 시작`, `개발 시작`
- **자동 판단 기준**: 사용자가 선택하지 않은 경우 작업 범위에 따라 적절한 방식 권장
- **안티패턴 정의**: 바로 구현 방안으로 넘어가는 것 금지

### Orchestrator - 작업 방식 선택 라우팅

- **라우팅 테이블 추가**: "작업 방식 선택" 의도를 `advisor`로 라우팅
- **Teacher vs Advisor 위임 조건 업데이트**: 작업 방식 선택 패턴을 Advisor 위임 대상에 추가
