# 0.33.0 (2025-12-01)

## Added

### 이슈 템플릿 테스트 요구사항 섹션

모든 이슈 생성/완료 Skill에 테스트 요구사항 섹션 추가

- **엔지니어 테스트**: speckit.implement 단계에서 테스트케이스로 작성하여 하나씩 검증
- **QA 테스트**: QA가 수동으로 검증할 수 있는 Step/Action/Expected Result 테이블

## Changed

### fast-track/references/issue-template.md

- 이슈 본문 템플릿에 `🧪 테스트 요구사항` 섹션 추가
- gh CLI 명령어 예시에 테스트 요구사항 포함
- 커밋 메시지 템플릿에도 테스트 요구사항 반영

### complete-draft-task/references/issue-template.md

- Draft → Complete Task 변환 시 테스트 요구사항 섹션 포함
- Before/After 템플릿 모두 업데이트
- Acceptance Criteria 다음에 테스트 요구사항 배치

### create-issues/SKILL.md

- Issue Format → Body 섹션에 테스트 요구사항 형식 추가
- 엔지니어 테스트: 체크박스 기반 테스트 케이스
- QA 테스트: 표 형식 유스케이스

## Impact

- 모든 신규 이슈에 테스트 요구사항 기재
- 엔지니어: TDD 기반 구현 가이드 확보
- QA: 명확한 검증 시나리오 확보
