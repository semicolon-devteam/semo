# 0.21.0

> Release Date: 2025-11-29

## Summary

Slack 통합 기능 추가 - notify-slack Skill 및 /SAX:slack 커맨드

## Added

### notify-slack Skill (신규)

| 파일 | 설명 |
|------|------|
| `skills/notify-slack/SKILL.md` | Slack 알림 Skill 메인 문서 |
| `skills/notify-slack/references/slack-id-mapping.md` | GitHub ID ↔ Slack ID 매핑 테이블 |
| `skills/notify-slack/references/message-templates.md` | Slack Block Kit 메시지 템플릿 |
| `skills/notify-slack/references/channel-config.md` | 대상 채널 설정 및 권한 관리 |

**기능**:
- Issue 생성 완료 시 Slack 채널(`#_협업`)에 자동 알림
- 커스텀 메시지 전송 지원 (/SAX:slack 커맨드)
- Spec/Epic 정보, Issue 목록, 담당자 멘션, GitHub 링크 포함
- Semicolon Notifier 앱 Bot Token 활용

### /SAX:slack 커맨드 (신규)

| 파일 | 설명 |
|------|------|
| `commands/SAX/slack.md` | Slack 메시지 전송 커맨드 |

**사용 예시**:

```bash
/SAX:slack #cm-land 채널에 'Roki' 멘션해서 #520번 이슈카드 확인해달라고 메세지 보내줘
```

- `chat:write`: 메시지 전송
- `chat:write.public`: 공개 채널 메시지 전송
- `users:read`: 사용자 ID 조회 (멘션용)
