## Summary

Slack 메시지 전송 커맨드 추가 - /SAX:slack

## Added

### /SAX:slack 커맨드 (신규)

| 파일 | 설명 |
|------|------|
| `commands/SAX/slack.md` | Slack 메시지 전송 커맨드 |

**기능**:

- Slack 채널에 커스텀 메시지 전송
- 팀원 멘션 지원 (@이름 또는 @github-id)
- GitHub 이슈/PR 링크 자동 포함
- Semicolon Notifier 앱 활용

**사용 예시**:

```bash
/SAX:slack #cm-land 채널에 'Roki' 멘션해서 #520번 이슈카드 확인해달라고 메세지 보내줘
```

### 필수 Slack 권한

- `chat:write`: 메시지 전송
- `chat:write.public`: 공개 채널 메시지 전송
- `users:read`: 사용자 ID 조회 (멘션용)
