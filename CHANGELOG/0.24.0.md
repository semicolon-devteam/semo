# 0.24.0 (2025-12-03)

- GitHub Projects 타입 필드 자동 설정 기능

  - `create-epic`: Epic 이슈 생성 시 타입을 "에픽"으로 자동 설정
- Projects 연동 실패 시 에러 검증 및 안내 메시지 추가
