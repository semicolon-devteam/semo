## Added

### Skills

- **report-bug**: QA 테스트 중 발견된 버그를 서비스 레포에 이슈카드로 생성
  - 테스트 케이스 연동 지원 (TC-{번호})
  - 태스크카드 형태의 버그 이슈 생성
  - 브랜치명 자동 제안 (`fix/{issue-number}-{slug}`)
  - 심각도(Critical/High/Medium/Low) 분류 + QA 대응 가이드
  - `qa-found` 라벨 자동 부여
  - references: issue-template.md, severity-guide.md
