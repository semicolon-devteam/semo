# 0.26.0

> complete-draft-task Skill 개선: Projects 이슈관리 보드 연결 확인 Phase 추가

## Changed

- **complete-draft-task Skill**: Phase 0 추가 - Projects 이슈관리 보드 연결 확인
  - Draft Task 작업 전 필수 선행 조건으로 Projects 연결 상태 확인
  - 미연결 시 자동으로 이슈관리 보드 (#1) 연결
  - GraphQL API를 통한 연결 상태 확인 및 추가 로직
  - Output Format에 Projects 보드 연결 상태 표시 추가

## Features

### Phase 0: Projects 이슈관리 보드 확인

- `gh api graphql`을 통한 Issue의 Projects 연결 상태 조회
- 연결 상태별 처리:
  - Projects 연결 없음 → 자동 연결
  - 다른 Project만 연결 → 이슈관리 보드 추가 연결
  - 이슈관리 보드 연결됨 → Phase 1로 진행
- 자동 연결 시 Organization Projects #1 ('이슈관리') 사용

## Usage

```javascript
// Draft Task 완성 시 자동으로 Projects 보드 확인
skill: complete-draft-task({
  draftIssues: [123, 124],
  epic: 144
});

// Phase 0에서 미연결 Issue 자동 처리
// → Projects 이슈관리 보드에 연결 후 Phase 1 진행
```
