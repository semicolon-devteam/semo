# 0.25.0 (2024-11-29)

## Changed

### GitHub Project 상태 조회 방식 개선

- **project-status.md**: 하드코딩된 상태 목록 제거, GitHub Project API 직접 조회 방식으로 변경
- **verification-steps.md**: 상태 목록을 직접 조회하도록 수정, project-status.md 참조 추가
- **task-progress.md (command)**: 상태 목록을 API 조회 방식으로 변경
- **task-progress/SKILL.md**: SoT 참조 추가

### 변경 사유

- 실제 GitHub Project 설정과 문서 불일치 문제 해결
- 상태 변경 시 문서 수정 불필요
- Single Source of Truth 원칙 적용

## Notes

- GitHub Project 상태는 `이슈관리` Project(번호: 1)에서 직접 조회
- 조회 명령어: `gh api graphql -f query='...'`
