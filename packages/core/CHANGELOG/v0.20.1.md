## Fixed

### sax-architecture-checker: Windows 심링크 폴백 지원

Windows 환경에서 심링크 생성 실패 시 파일 복사로 자동 폴백하도록 개선:

- `create_link_or_copy()` 함수 추가: 심링크 실패 시 파일/디렉토리 복사로 대체
- `detect_os()` 함수 추가: Windows/macOS/Linux 환경 감지
- `print_fallback_warning()` 함수 추가: Windows 환경 안내 메시지 출력
