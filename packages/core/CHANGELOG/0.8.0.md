# 0.8.0 (2025-12-01)

## Changed

### notify-slack 릴리스 알림 템플릿 표준화

릴리스 알림 메시지 형식을 표준화하여 일관된 포맷으로 전송

#### 표준 메시지 형식

```text
🚀 SEMO 패키지 업데이트

패키지             버전
semo-meta          v0.30.0

변경 내역
• 변경사항 1
• 변경사항 2

🔗 GitHub
```

### message-templates.md 개선

- 메시지 미리보기 섹션 추가
- curl 명령어 예시 추가 (변수 포함)
- 변경 내역 작성 규칙 테이블 추가
- 복수 패키지 릴리스 템플릿 추가

### SKILL.md 업데이트

- 표준 메시지 형식 예시 추가
- 메시지 구조 테이블 추가
- message-templates.md 참조 링크 추가

## Impact

- 모든 SEMO 패키지 릴리스 알림이 동일한 형식으로 전송됨
- 변경 내역이 bullet point 형식으로 가독성 향상
