# v0.24.0

> 2025-12-10

## Added

### Commands
- `/SAX:onboarding` - SAX 통합 온보딩 프로세스 명령어 신설
  - Phase 0-2: 환경 진단, 조직 참여 확인, SAX 학습 (공통)
  - Phase 3: 설치된 패키지의 `skill:onboarding-{package}` 자동 호출
  - Phase 4: 온보딩 완료 및 메타데이터 저장

### Skills
- `onboarding` - SAX 통합 온보딩 프로세스 스킬
  - 모든 패키지에서 공통으로 사용
  - 패키지 자동 감지 및 해당 온보딩 스킬 위임
- `memory` - 세션 간 컨텍스트 영속화 스킬
  - 아키텍처 결정 사항 저장
  - 팀 선호도 기록
  - "기억 상실" 문제 해결

## Changed

### Commands
- `/SAX:health` - 토큰 리포트 기능 추가 (Phase 3)
  - CLAUDE.md, PRINCIPLES.md 등 토큰 사용량 분석
  - 세션 초기화 비용 예상치 표시
  - 메모리 상태 확인 (Phase 4)

## Migration

기존 패키지별 `/SAX:onboarding` 명령어는 DEPRECATED 처리되었습니다.
sax-core의 통합 `/SAX:onboarding`을 사용하세요.

```text
[기존] 패키지별 /SAX:onboarding → onboarding-master Agent
[신규] 통합 /SAX:onboarding → sax-core/skill:onboarding → skill:onboarding-{package}
```
