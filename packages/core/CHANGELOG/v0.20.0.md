# v0.20.0

**Release Date**: 2025-12-05

## Changes

### Changed
- `/SEMO:health` 커맨드 기능 확장
  - Phase 1: 개발 환경 검증 (패키지별 `skill:health-check` 호출)
  - Phase 2: SEMO 구조 검증 (`semo-architecture-checker` 호출)
- `/SEMO:health-check` 하위호환 지원 (동일 기능으로 라우팅)

## Details

### 통합된 health 커맨드

기존에 각 패키지에 산재되어 있던 `health-check` 커맨드를 semo-core의 `health` 커맨드로 통합:

| 기존 | 변경 후 |
|------|---------|
| `/SEMO:health-check` (패키지별) | `/SEMO:health` (semo-core 통합) |
| 개발 환경만 검증 | 환경 + SEMO 구조 통합 검증 |

### 호출 흐름

```
/SEMO:health
  ├─ Phase 1: skill:health-check (패키지별, 있을 경우)
  └─ Phase 2: skill:semo-architecture-checker (공통)
```
