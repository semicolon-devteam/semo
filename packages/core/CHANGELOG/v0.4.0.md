# v0.4.0

> compliance-checker 검증 규칙 확장

## Added

- **validation-rules.md**: 버저닝 완료 검증 규칙 추가
  - Agent/Skill/Command 변경 시 VERSION 파일 변경 여부 검증
  - CHANGELOG 생성 여부 검증
  - 버저닝 필요 파일 패턴 정의
  - TodoWrite 자동 추가 규칙

- **validation-rules.md**: 세션 시작 시 version-updater 검증 규칙 추가
  - 새 세션 + SEMO 설치 시 version-updater 호출 여부 확인
  - 위반 시 CRITICAL 레벨 경고

## Changed

- **위반 심각도 정의** 업데이트
  - CRITICAL: 세션 시작 시 version-updater 미호출 추가
  - WARNING: 버저닝 미완료 상태에서 커밋 시도 추가

## Notes

이 버전부터 compliance-checker가 버저닝 완료 여부와 version-updater 호출 여부를 검증합니다.
