# 0.9.0 (2025-12-01)

## Added

- **feedback Skill**: 공통 피드백 수집 스킬 추가
  - 모든 SEMO 패키지에서 공통으로 사용
  - 패키지명 파라미터로 대상 레포지토리 지정
  - 버그 리포트 및 개선 제안 이슈 자동 생성
  - 암시적/명시적 트리거 지원

- **feedback Command**: `/SEMO:feedback` 공통 커맨드 추가
  - 현재 컨텍스트의 패키지 자동 감지
  - 불명확 시 사용자에게 패키지 선택 질문

## Changed

- 기존 semo-po, semo-next의 feedback 스킬/커맨드를 core로 통합
- 각 패키지는 core의 feedback을 참조하도록 변경
