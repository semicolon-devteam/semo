# 0.10.0 (2025-12-02)

## Added

- **공통 커맨드 추가**
  - `/SEMO:help` - SEMO 도움말 및 Semicolon 팀 컨텍스트 응답
  - `/SEMO:slack` - Slack 메시지 전송 (notify-slack 스킬 호출)
  - `/SEMO:update` - SEMO 패키지 버전 업데이트 (version-updater 스킬 호출)

- **semo-help 스킬 추가**
  - SEMO 사용법, 패키지 정보, 팀 컨텍스트 질문 응답
  - docs 레포 연동 가이드 포함
  - 패키지 정보 조회 유틸리티 포함

## Changed

- 공통 커맨드를 semo-core에서 중앙 관리
- install-sax.sh에서 commands/SEMO 병합 지원 추가

## Notes

- 모든 패키지에서 `/SEMO:help`, `/SEMO:slack`, `/SEMO:update`, `/SEMO:feedback` 공통 사용 가능
- 패키지별 특화 커맨드는 각 패키지에서 정의하며, install-sax.sh가 병합
