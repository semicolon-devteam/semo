# v0.1.0 (2024-11-27)

## Initial Release

SEMO-PO 패키지 최초 릴리스 (PO/기획자용)

### Added

#### Agents
- `orchestrator` - 요청 라우팅
- `epic-master` - Epic 생성
- `draft-task-creator` - Draft Task 생성
- `spec-writer` - Spec 초안 작성
- `onboarding-master` - 신규 사용자 온보딩
- `teacher` - 학습 안내

#### Skills
- `health-check` - 개발 환경 검증
- `assign-project-label` - 프로젝트 라벨 및 Projects 연결
- `detect-project-from-epic` - Epic 프로젝트 라벨 감지
- `check-backend-duplication` - core-backend 중복 체크
- `assign-estimation-point` - Estimation Point 할당
- `generate-acceptance-criteria` - AC 자동 생성
- `create-design-task` - 디자인 Task 생성
- `validate-task-completeness` - Draft Task 필수 항목 검증
- `auto-label-by-scope` - Epic 범위 기반 자동 라벨링
- `estimate-epic-timeline` - Epic 전체 일정 예측
- `check-team-codex` - 팀 규칙 검증
- `create-epic` - Epic 생성

#### Commands
- `/SEMO:onboarding` - 신규 PO/기획자 온보딩
- `/SEMO:health-check` - 개발 환경 검증
- `/SEMO:help` - 대화형 도우미
