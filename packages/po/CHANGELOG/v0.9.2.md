# v0.9.2

> 2025-11-28

## Fixed

### 심링크 경로 수정 (.claude/SAX/commands → .claude/commands/SAX)

Claude Code는 `.claude/commands/SAX/*.md` 경로에서 `/SEMO:*` 명령어를 인식합니다.

**변경된 파일**:

- sax-update Skill: 심링크 생성 경로 수정
- health-check validation-items: 검증 경로 수정
- version-updater Agent: 심링크 경로 수정

**올바른 심링크 구조**:

```text
.claude/
├── CLAUDE.md → semo-po/CLAUDE.md
├── agents → semo-po/agents
├── skills → semo-po/skills
└── commands/
    └── SEMO → ../semo-po/commands
```

**명령어 인식 예시**:

- `.claude/commands/SAX/help.md` → `/SEMO:help`
- `.claude/commands/SAX/health-check.md` → `/SEMO:health-check`
