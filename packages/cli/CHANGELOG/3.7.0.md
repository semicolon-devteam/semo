# v3.7.0

> 2025-12-30

## Added

- **`semo doctor` 명령어**: MCP 및 GitHub 설정 문제 진단
  - gh CLI 설치 및 인증 상태 확인
  - GitHub 토큰 환경변수 설정 상태 확인
  - MCP 서버 등록 상태 확인
  - settings.json 설정 검증
  - 문제 발견 시 해결 방법 안내

- **GitHub MCP 토큰 설정 안내**: `semo init` 완료 시 토큰 설정 가이드 표시
  - 환경변수 미설정 시 경고 및 설정 명령어 안내
  - gh CLI 인증 상태 기반 토큰 자동 감지

## Changed

- **GitHub MCP 서버 설정**: 환경변수 `GITHUB_PERSONAL_ACCESS_TOKEN` 추가
  - `@modelcontextprotocol/server-github`는 환경변수로 토큰 전달 필요
  - gh CLI 토큰을 환경변수로 export하여 사용

## Usage

```bash
# 설정 문제 진단
semo doctor

# GitHub MCP 토큰 설정 (일회성)
export GITHUB_PERSONAL_ACCESS_TOKEN="$(gh auth token)"

# 영구 설정 (셸 재시작 후 적용)
echo 'export GITHUB_PERSONAL_ACCESS_TOKEN="$(gh auth token)"' >> ~/.zshrc
```
