# v3.8.0

> 2026-01-02

## Changed

- **`semo doctor` 명령어 개선** (#116)
  - semo-mcp 체크 로직을 v3.0 아키텍처에 맞게 업데이트

## Added

- **CLI 도구 체크 섹션 추가** (5-1. CLI 도구)
  - `gh` (GitHub CLI) 설치 여부 확인
  - `supabase` (Supabase CLI) 설치 여부 확인
  - Slack은 curl 사용 안내

## Migration

- v3.0 아키텍처: 스킬에서 CLI 직접 호출 방식
  - GitHub 작업: `gh issue create`, `gh pr create`
  - Supabase 작업: `supabase` CLI
  - Slack 작업: `curl -H "Authorization: Bearer $SLACK_BOT_TOKEN"`
