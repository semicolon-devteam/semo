# v3.9.0 (2026-01-03)

## Fixed

- **Windows 심링크 호환성 개선** (#118, #117)
  - Git Bash, MSYS, Cygwin 환경에서도 Windows로 올바르게 감지
  - Junction 생성 로직 개선: 폴더는 Junction, 파일은 복사
  - Junction 실패 시 자동으로 xcopy 복사로 fallback
  - VSCode/탐색기에서 심링크 파일 정상 인식

## Changed

- `isWindows` 판별 로직 확장
  - `os.platform() === "win32"` 외에도 `OSTYPE`, `TERM_PROGRAM` 환경변수 확인
  - mintty (Git Bash 기본 터미널) 환경 지원

## Technical Details

- `createSymlinkOrJunction` 함수 리팩토링
  - 파일/폴더 타입에 따른 분기 처리
  - `windowsHide: true` 옵션으로 cmd 창 숨김
  - 절대 경로 사용으로 경로 형식 일관성 보장
