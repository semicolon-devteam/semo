# v3.6.0

> 2025-12-30

## Changed

- **패키지 구조 통합**: Business/Engineering/Operations 레이어 분류 제거
  - `semo init`에서 패키지 선택 프롬프트 제거 (기본 설치만 진행)
  - `semo list`에서 그룹별 표시 대신 단순 목록 표시
  - `semo add`에서 그룹 일괄 설치 안내 제거

- **EXTENSION_PACKAGES 단순화**
  - 기존 12개 레이어별 패키지 → 3개 Extension으로 통합 (meta, semo-hooks, semo-remote)
  - 불필요한 그룹 관련 코드 제거 (PACKAGE_GROUPS, getPackagesByGroup 등)

## Removed

- 프로젝트 유형 자동 감지 기능 제거 (detectProjectType)
- 레이어별 패키지 정의 제거 (biz/*, eng/*, ops/*)
- 그룹 일괄 설치 기능 제거 (semo add biz, semo add eng 등)

## Migration

- 기존에 `biz/*`, `eng/*`, `ops/*` 패키지를 사용하던 프로젝트는 직접 `semo-system/` 디렉토리에서 필요한 파일을 관리하세요
- 새로운 Extension은 `semo add meta`, `semo add semo-hooks`, `semo add semo-remote`로 설치 가능
