# 1.2.0 (2025-12-18)

## Added

- **semicolon-reviewer Phase 7**: PR 리뷰 등록 기능 추가
  - 코드 리뷰 완료 후 GitHub PR에 리뷰 코멘트 자동 작성
  - Critical/Warning 개수에 따라 APPROVE/COMMENT/REQUEST_CHANGES 자동 결정
  - `gh pr review` 명령어로 PR 리뷰 등록

- **change-to-testing 라우팅**: orchestrator에 테스트 요청 스킬 라우팅 추가
  - "테스트 요청", "QA에 넘겨", "테스트중으로" 키워드로 호출 가능
  - QA 담당자(@kokkh) 자동 할당

## Changed

- **review-phases.md**: Phase 7 (PR 리뷰 등록) 상세 문서 추가

## Fixed

- #37: 개발자 환경에서 "테스트 요청" 시 change-to-testing 스킬이 라우팅되지 않던 문제
- #38: semicolon-reviewer가 PR에 리뷰를 등록하지 않던 문제
