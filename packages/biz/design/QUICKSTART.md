# biz/design Quickstart

> UI/UX 디자이너를 위한 빠른 시작 가이드

## 대상

- UI/UX 디자이너
- 컨셉 설계 담당자

## 주요 스킬

| 스킬 | 설명 | 트리거 예시 |
| ---- | ---- | ----------- |
| `generate-mockup` | 목업 생성 | "목업 만들어줘" |
| `design-handoff` | 디자인 핸드오프 | "개발팀에 전달해줘" |

## 빠른 시작 예시

```text
"로그인 화면 목업 만들어줘"   → skill:generate-mockup
"개발팀에 디자인 전달해줘"    → skill:design-handoff
```

## 상세 튜토리얼

```text
"디자인 온보딩 실습해줘" → skill:onboarding-design
```
