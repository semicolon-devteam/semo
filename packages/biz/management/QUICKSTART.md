# biz/management Quickstart

> PM/프로젝트 관리자를 위한 빠른 시작 가이드

## 대상

- PM (Project Manager)
- 스프린트 관리자
- 일정/인력 관리 담당자

## 주요 스킬

| 스킬 | 설명 | 트리거 예시 |
| ---- | ---- | ----------- |
| `create-sprint` | Sprint 생성 | "새 스프린트 만들어줘" |
| `assign-to-sprint` | Task 할당 | "Task 스프린트에 할당해줘" |
| `generate-progress-report` | 진행 리포트 | "진행도 리포트 만들어줘" |
| `calculate-velocity` | 벨로시티 계산 | "팀 벨로시티 계산해줘" |

## 빠른 시작 예시

```text
"새 스프린트 만들어줘"        → skill:create-sprint
"Task 스프린트에 할당해줘"    → skill:assign-to-sprint
"진행도 리포트 만들어줘"      → skill:generate-progress-report
"팀 벨로시티 계산해줘"        → skill:calculate-velocity
```

## Sprint 워크플로우

```text
1. Sprint 생성
   → "새 스프린트 만들어줘"
   → 2주 단위 Sprint 생성

2. Task 할당
   → "Task #123 이번 스프린트에 할당해줘"
   → 담당자, 포인트 지정

3. 진행도 추적
   → "진행도 리포트 만들어줘"
   → 번다운 차트, 완료율

4. Sprint 종료
   → "스프린트 종료해줘"
   → 회고, 다음 계획
```
