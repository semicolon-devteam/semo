# 1.5.0 (2025-12-28)

## Changed

- **스킬명 변경**: `generate-meeting-template` → `generate-meeting-minutes`
  - "minutes"가 회의록의 국제적 표준 용어
  - 폴더 및 frontmatter name 업데이트

- **Orchestrator 라우팅 우선순위 개선**
  - "정기회의록" 키워드가 "회의록" 보다 높은 우선순위
  - 회의록 관련 스킬 라우팅 규칙 명확화:
    - `generate-meeting-minutes`: 새 Discussion 생성 (정기회의록, N월 N/N, 회의록 만들어줘)
    - `summarize-meeting`: 기존 Discussion에 내용 작성 (STT, 회의 정리)

## Fixed

- **라우팅 충돌 해결** (#93)
  - "정기회의록 만들어줘" 요청이 `summarize-meeting`으로 잘못 라우팅되던 버그 수정
