# v0.5.0 (2024-11-27)

## Added

### Orchestrator

- **Git 명령 인터셉트 기능 추가**
  - `git commit`, `git push` 명령 자동 감지 → `skill:git-workflow`로 라우팅
  - 한글 커밋 요청 ("커밋해줘", "푸시해줘") 감지
  - `--no-verify` 플래그 감지 시 즉시 차단 및 안내 메시지 출력
  - Routing Decision Table에 Git 커밋/푸시 라우팅 규칙 추가

### skill:git-workflow

- **--no-verify 차단 로직 강화**
  - `--no-verify` 또는 `-n` 플래그 감지 시 커밋 즉시 중단
  - 차단 사유 및 해결 방법 안내 메시지 출력
  - 예외 없음 정책 명시 (사용자 요청, 긴급 상황에서도 거부)
