## Added

### spec Skill Phase 0 Brainstorming

obra/superpowers의 brainstorming 방법론을 spec Skill에 Phase 0으로 통합:

- **자동 감지 활성화**: 모호한 아이디어 표현 감지 시 자동으로 Brainstorming 모드 진입
- **3단계 워크플로우**: 아이디어 이해 → 접근 방식 탐색 → 디자인 합의
- **단일 질문 원칙**: 한 번에 하나의 질문으로 사용자 부담 최소화
- **객관식 옵션 제공**: 2-3가지 선택지로 의사결정 단순화
- **spike Skill 연계**: 기술적 불확실성 발견 시 spike 전환 제안

**활성화 트리거**:
- "아이디어가 있는데"
- "뭔가 만들고 싶어"
- "이런 거 되나"
- "어떻게 하면 좋을까?"

**Phase Flow 변경**:
```
brainstorm? → specify → clarify? → plan → checklist? → tasks → issues? → report
```

**참조 문서**:
- `skills/spec/SKILL.md` - Skill 정의 (Phase 0 추가)
- `skills/spec/references/brainstorming-guide.md` - Brainstorming 상세 가이드
- `skills/spec/references/phase-details.md` - Phase 0 상세 추가
