# SEMO-Next v0.16.0 - 2025-11-28

## docx Skill 추가

### Added

- **docx Skill** (semo-next)
  - Word 문서 생성, 편집, 변환 통합 Skill
  - Anthropic Skills 표준 기반 포팅
  - Use cases:
    - spec.md → docx 변환 (외부 공유용)
    - 릴리스 노트 Word 생성
    - 기술 문서 docx 내보내기
    - 기존 docx 편집 (Tracked Changes)

- **references/**
  - `docx-js.md` - JavaScript 문서 생성 가이드
  - `ooxml.md` - Python 문서 편집 가이드
  - `workflow-examples.md` - SEMO 워크플로우 연동 예시

- **scripts/ooxml/**
  - `unpack.py` - docx 압축 해제
  - `pack.py` - docx 압축
  - `validate.py` - XML 검증

### Dependencies

- `pandoc` - Markdown ↔ Word 변환
- `docx` (npm) - JavaScript 문서 생성
- `defusedxml` (pip) - 안전한 XML 파싱
