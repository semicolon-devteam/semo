# v0.11.0

> 2025-11-28

## Added

### fast-track Skill

개발자 주도의 경미한 수정을 Epic→Task 프로세스 없이 빠르게 처리하는 새 Skill 추가:

- **적격성 자동 검사**: 4대 조건 (영향 범위, 기능 변경, 테스트 영향, 소요 시간) 체크
- **사후 이슈 생성**: 작업 완료 후 GitHub Issue 자동 생성으로 PO 가시성 확보
- **카테고리 분류**: 오타/문구, 스타일, 핫픽스, 코드 정리 4가지 카테고리

**대상 작업**:
- 오타/문구 수정 (`typo`)
- 스타일 미세조정 (`style`)
- 버그 핫픽스 (`hotfix`)
- 코드 정리 (`chore`)

**참조 문서**:
- `skills/fast-track/SKILL.md` - Skill 정의
- `skills/fast-track/references/eligibility-check.md` - 적격성 체크 가이드
- `skills/fast-track/references/issue-template.md` - 이슈 템플릿
- `skills/fast-track/references/category-guide.md` - 카테고리 가이드
