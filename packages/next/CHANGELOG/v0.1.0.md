# v0.1.0 (2024-11-27)

## Initial Release

SEMO-Next 패키지 최초 릴리스 (Next.js 개발자용)

### Added

#### Agents
- `orchestrator` - 요청 라우팅
- `spec-master` - SDD Phase 1-3
- `database-master` - DB 및 Supabase 통합
- `advisor` - 조언 제공
- `teacher` - 학습 안내
- `onboarding-master` - 신규 개발자 온보딩
- `ddd-architect` - DDD 아키텍처 설계
- `implementation-master` - 구현 관리
- `migration-master` - 마이그레이션 관리
- `quality-master` - 품질 관리
- `semicolon-reviewer` - 코드 리뷰
- `spike-master` - Spike 관리

#### Skills
- `health-check` - 개발 환경 검증
- `task-progress` - 워크플로우 진행도 확인
- `spec` - SDD 명세 워크플로우
- `implement` - ADD 구현 워크플로우
- `verify` - Phase 5 종합 검증
- `check-team-codex` - 팀 코덱스 검증
- `validate-architecture` - DDD 아키텍처 검증
- `scaffold-domain` - 도메인 구조 생성
- `fetch-supabase-example` - Supabase 패턴 참조
- `git-workflow` - Git 워크플로우 자동화
- `create-issues` - GitHub Issues 생성
- `project-kickoff` - 프로젝트 시작 가이드
- `migration-analyzer` - 마이그레이션 분석
- `constitution` - 프로젝트 헌법
- `fetch-api-spec` - API 스펙 참조
- `fetch-team-context` - 팀 컨텍스트 참조
- `project-context` - 프로젝트 컨텍스트
- `spike` - Spike 실행

#### Commands
- `/SEMO:onboarding` - 신규 개발자 온보딩
