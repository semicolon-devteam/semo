# 0.32.0 (2025-12-01)

## Added

### Spec-First Branching Workflow

- **새로운 워크플로우**: dev 브랜치에서 Spec 작성 → 원격 푸시 → Feature 브랜치 생성
- **목적**: 다른 작업자도 특정 도메인의 Spec을 공유받을 수 있도록 함
- Feature 브랜치에서는 실제 코드 구현만 수행
