# 1.9.0

> SAX 커맨드 추가 (SuperClaude 대응)

## Added

### Commands

- `/SAX:debug` - 버그/에러 분석 (debug-master Agent 호출)
- `/SAX:context` - 컨텍스트 빠른 파악 (load-context Skill 호출)
- `/SAX:improve` - 코드 개선/리팩토링 (improve-code Skill 호출)
- `/SAX:test` - 테스트 실행 및 커버리지 (run-tests Skill 호출)
- `/SAX:analyze` - 코드 종합 분석 (analyze-code Skill 호출)

## SuperClaude 대응 매핑

| SAX Command | SuperClaude Command | 호출 대상 |
|-------------|---------------------|----------|
| `/SAX:debug` | `/sc:troubleshoot` | debug-master Agent |
| `/SAX:context` | `/sc:load` | load-context Skill |
| `/SAX:improve` | `/sc:improve` | improve-code Skill |
| `/SAX:test` | `/sc:test` | run-tests Skill |
| `/SAX:analyze` | `/sc:analyze` | analyze-code Skill |

## Notes

- 모든 커맨드는 Spring Boot + Kotlin + WebFlux 환경에 최적화
- P0~P2에서 구현된 Agent/Skill에 대한 빠른 접근 제공
