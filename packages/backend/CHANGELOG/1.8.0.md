# 1.8.0

> 2025-12-04

## Added

- **analyze-code Skill** (skills/)
  - 코드 종합 분석 Skill (품질/보안/성능/아키텍처)
  - 4가지 Focus 지원: quality, security, performance, architecture
  - 점수 기반 평가 시스템 (0-100점)
  - Reference: analysis-patterns.md, severity-definitions.md, score-calculation.md

## Changed

- **quality-master Agent** (agents/)
  - Extended Analysis Mode 추가 (`--analyze` 플래그)
  - analyze-code, run-tests Skill 연동
  - Multi-focus 종합 분석 워크플로우
  - 통합 리포트 출력 (점수 대시보드 + 이슈 목록)

- **orchestrator Agent** (agents/)
  - 라우팅 테이블에 analyze-code 추가
  - Fast Track 라우팅 경로 확장
  - 키워드 기반 라우팅 확장: 분석, 스캔, 취약점, 보안 검사

### Migration Guide

**sax-backend 사용자**:

1. 코드 분석 요청 시 자동으로 `skill:analyze-code`로 라우팅됩니다
   - 키워드: 분석, 스캔, 취약점, 보안 검사
2. quality-master에서 `--analyze` 플래그로 종합 분석 활성화 가능
3. 기존 Phase 5 검증 워크플로우는 영향 없음
