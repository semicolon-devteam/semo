# 1.6.0

> 2025-12-04

## Added

- **debug-master Agent** (agents/)
  - 버그 수정 및 에러 분석 전담 Agent
  - 5단계 진단 워크플로우 (증상 수집 → 컨텍스트 분석 → 원인 파악 → 수정 제안 → 수정 적용)
  - Spring Boot + Kotlin + WebFlux 환경 최적화
  - Reference: diagnostic-workflow.md, common-errors.md, fix-patterns.md

- **load-context Skill** (skills/)
  - 도메인/기능 컨텍스트 빠른 파악 Skill
  - 5가지 컨텍스트 소스 지원 (specs/, domain/, git history, GitHub issues, tests)
  - Fast Track 워크플로우 지원
  - Reference: context-sources.md, output-format.md

## Changed

- **orchestrator Agent** (agents/)
  - 라우팅 테이블에 debug-master, load-context 추가
  - Fast Track 라우팅 경로 명시
  - 키워드 기반 라우팅 확장: 에러, 버그, 문제, 디버그, 컨텍스트, 파악해

### Migration Guide

**semo-backend 사용자**:

1. 버그/에러 관련 요청 시 자동으로 `debug-master` Agent로 라우팅됩니다
2. 컨텍스트 파악 요청 시 자동으로 `skill:load-context`로 라우팅됩니다
3. 기존 SDD 워크플로우는 영향 없음 (Fast Track으로 분류)
