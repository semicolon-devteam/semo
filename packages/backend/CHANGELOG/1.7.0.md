# 1.7.0

> 2025-12-04

## Added

- **improve-code Skill** (skills/)
  - 코드 품질, 성능, 유지보수성 개선 Skill
  - 4가지 개선 유형: quality, performance, reactive, security
  - Kotlin/Spring Boot/WebFlux 환경 최적화
  - Reference: code-smells.md, improvement-patterns.md, validation-checklist.md

- **run-tests Skill** (skills/)
  - 테스트 실행 및 품질 검증 Skill
  - 4가지 테스트 유형: unit, integration, all, specific
  - JaCoCo 커버리지 분석 지원
  - Reference: test-patterns.md, coverage-guide.md, troubleshooting.md

## Changed

- **orchestrator Agent** (agents/)
  - 라우팅 테이블에 improve-code, run-tests 추가
  - Fast Track 라우팅 경로 확장
  - 키워드 기반 라우팅 확장: 개선, 리팩토링, 품질, 최적화, 테스트, 검증해

### Migration Guide

**semo-backend 사용자**:

1. 코드 개선 요청 시 자동으로 `skill:improve-code`로 라우팅됩니다
   - 키워드: 개선, 리팩토링, 품질, 최적화
2. 테스트 실행 요청 시 자동으로 `skill:run-tests`로 라우팅됩니다
   - 키워드: 테스트, 검증해, 확인해
3. 기존 워크플로우는 영향 없음 (Fast Track으로 분류)
