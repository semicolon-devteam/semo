# 0.1.0 (2025-12-11)

## Added

### Agents
- `orchestrator`: MVP 작업 라우팅 및 의도 분석
- `mvp-architect`: DDD 4-layer 아키텍처 + metadata 확장 패턴 설계
- `implementation-master`: Phase-gated MVP 구현 (간소화된 ADD)
- `onboarding-master`: 6단계 MVP 개발자 온보딩

### Skills
- `health-check`: 환경 및 MCP 서버 검증
- `sync-interface`: core-interface JSON artifacts 타입 동기화
- `scaffold-mvp-domain`: DDD 4-layer 도메인 구조 생성
- `supabase-fallback`: Spring Boot 미가동 시 Supabase GraphQL 직접 쿼리
- `implement-mvp`: Phase-gated MVP 구현
- `verify-integration`: 커뮤니티 솔루션 통합 호환성 검증

### Commands
- `/SAX:onboarding`: MVP 개발자 온보딩
- `/SAX:health`: 환경 검증
- `/SAX:scaffold`: 도메인 구조 생성
- `/SAX:implement`: 구현 시작
- `/SAX:mockup`: Antigravity 목업 위임
- `/SAX:verify`: 통합 검증

### Antigravity Integration
- `.agent/rules/`: SAX 원칙, DDD 패턴, 스키마 확장 전략
- `.agent/workflows/`: mockup, component, browser-test 워크플로우
