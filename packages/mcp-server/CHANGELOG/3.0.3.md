## Troubleshooting

Windows + VSCode Extension 환경에서 MCP 도구를 찾을 수 없는 경우:

1. VSCode/Claude Code 완전 재시작
2. `.claude/settings.json`에서 `npx` → `npx.cmd` 변경
3. `npm cache clean --force` 실행
