# v3.0.0

> 2026-01-02

## Breaking Changes

- **Slack/GitHub/Supabase 도구 제거** (#116)
  - 제거된 도구:
    - `slack_send_message`, `slack_lookup_user`, `slack_list_channels`, `slack_find_channel`
    - `github_create_issue`, `github_create_pr`
    - `supabase_query`
    - `semo_get_slack_token`
    - `semo_route`
  - 마이그레이션: 스킬에서 CLI 직접 호출 방식으로 전환
    - GitHub: `gh` CLI
    - Supabase: `supabase` CLI
    - Slack: `curl` + 환경변수 SLACK_BOT_TOKEN

## Changed

- **MCP 서버 역할 축소**
  - 이전: Slack/GitHub/Supabase/Memory/Remote 통합
  - 이후: Memory + Remote 기능만 제공
  - 이유: 보안 강화 (토큰 분산 관리), 유지보수 단순화

## Retained

- **SEMO Memory** (8개 도구)
  - `semo_remember`, `semo_recall`, `semo_save_fact`, `semo_get_facts`
  - `semo_get_history`, `semo_memory_status`
  - `semo_process_embeddings`, `semo_recall_smart`

- **SEMO Remote** (3개 도구)
  - `semo_remote_request`, `semo_remote_respond`, `semo_remote_pending`
