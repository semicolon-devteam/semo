# [0.4.0] - 2025-12-01

## Added

- **set-estimate Skill**: Task 작업량(Estimate Point)을 독립적으로 설정하는 새로운 Skill
  - Sprint 할당 없이 작업량만 설정 가능
  - 피보나치 수열 기반 포인트 (1, 2, 3, 5, 8, 13)
  - 13pt 이상 시 Task 분할 권장 안내
  - 백로그 그루밍, 계획 포커 워크플로우 지원

- **/SEMO:sprint estimate** 커맨드: 작업량 설정을 위한 CLI 인터페이스
  - 단일/복수 Task 작업량 일괄 설정
  - `--point` 옵션으로 작업량 지정

## Changed

- **sprint-master Agent**: set-estimate Skill 연동 추가
  - 호출 Skills 목록에 set-estimate 추가
  - Task 작업량 설정 워크플로우 추가

- **audit-issues Skill**: 작업량 미할당 Task에 대한 해결 안내 개선
  - set-estimate Skill 사용 안내 추가
  - Related 섹션에 set-estimate 링크 추가

- **orchestrator routing-table**: 작업량 관련 라우팅 규칙 추가
  - sprint-master 트리거 키워드: "작업량", "estimate", "point", "포인트", "추정"
  - Skill 직접 라우팅: set-estimate 추가

- **/SEMO:sprint** 커맨드 문서: estimate action 추가
  - Sprint 전체 워크플로우에 작업량 설정 단계 추가
  - 연관 Skills에 set-estimate 추가

## Workflow Integration

작업량 할당이 Sprint 워크플로우에 자연스럽게 통합됨:

```text
백로그 그루밍
    ↓
audit-issues (작업량 미할당 감지)
    ↓
set-estimate (작업량 설정)  ← NEW
    ↓
assign-to-sprint (Sprint 할당)
    ↓
Sprint 진행
```
