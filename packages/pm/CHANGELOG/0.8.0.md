# 0.8.0 (2025-12-02)

## Added

- `sync-iteration` Skill 추가
  - 미완료 이슈들의 Iteration을 현재(Current)로 일괄 동기화
  - Iteration이 없는 OPEN 이슈도 current로 설정
  - `--dry-run` 옵션으로 미리보기 지원
  - 트리거: "iteration 업데이트", "이터레이션 동기화"

- `/SAX:sprint sync` Command action 추가
  - `/SAX:sprint sync` - 실행
  - `/SAX:sprint sync --dry-run` - 미리보기만
