# v0.11.0

> 2025-11-27

## Added

### sax-help Skill 추가

- SAX-Meta 패키지 사용 가이드 제공
- `/SAX:help` Command와 연동
- 주요 명령어 및 워크플로우 안내

### /SAX:help Command 추가

- `skill:sax-help` 호출하여 도움말 표시
- Orchestrator에서 "도움말", "SAX란", "어떻게 해" 키워드로 자동 라우팅

### Multi-Package Workflow 문서 추가

- `agent-manager/references/multi-package-workflow.md`
- `skill-manager/references/multi-package-workflow.md`
- `command-manager/references/multi-package-workflow.md`
- 복수 패키지 접두사 명령 (`[po | next]`) 처리 가이드

## Changed

### Orchestrator Skill 호출 방법 명시

- Agent 위임과 Skill 호출 구분 명확화
- Skill 호출 시 `/` 슬래시로 메시지 블록 종료
- feedback Skill 호출 예시 추가

### SAX 오류 지적 키워드 명시화

- 기존: 일반적인 오류 표현 → SAX 관련 여부 판단 어려움
- 변경: `SAX가 왜`, `SAX 동작이`, `[SAX] 메시지가`, `SAX 결과가` 등 명시적 키워드
- 애매한 경우 확인 질문 로직 추가

### feedback Skill When to Use 섹션 추가

- Orchestrator → Skill 호출 조건 명시
- 감지 키워드 및 호출 흐름 문서화
- 애매한 경우 확인 질문 템플릿 추가
