# 0.33.0 (2025-12-02)

## Added

### version-manager: 피드백 이슈 연동 기능

피드백 기반 버저닝 시 피드백 작성자에게 알림하는 기능 추가

#### Phase 10 신규 단계

| 단계 | 내용 |
|------|------|
| 1 | 커밋 메시지에서 피드백 이슈 감지 (`#123`, `Fixes #`, `Closes #`) |
| 2 | 이슈 정보 조회 및 피드백 이슈 판별 |
| 3 | GitHub 이슈에 완료 코멘트 추가 (@작성자 멘션) |
| 4 | Slack 알림에 피드백 작성자 멘션 추가 |

#### 피드백 이슈 판별 조건

- `bug` 또는 `enhancement` 라벨 존재
- 본문에 `SAX Feedback Skill` 문구 포함

#### Input Schema 확장

```json
{
  "feedback_issues": [
    {
      "repo": "sax-po|sax-next|sax-meta|sax-core",
      "number": 123
    }
  ]
}
```

## Changed

- `references/workflow.md`: 9단계 → 10단계 프로세스로 확장
- Validation 섹션에 피드백 이슈 완료 처리 확인 항목 추가

## Impact

- sax-core의 feedback Skill로 수집된 피드백 반영 시 작성자에게 자동 알림
- 피드백 제공자의 기여 인정 및 커뮤니케이션 향상
- GitHub 이슈 기반 피드백 루프 완성
