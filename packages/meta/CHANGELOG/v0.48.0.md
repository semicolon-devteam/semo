# v0.48.0

**Release Date**: 2025-12-09

## Changed

### Orchestrator Cross-Package Routing 규칙 추가

다른 패키지의 전문 영역 요청 감지 시 해당 패키지로 인계 권유하는 규칙 추가:

#### 변경된 파일

| 파일 | 변경 내용 |
|------|----------|
| `agents/orchestrator/orchestrator.md` | Cross-Package Routing 섹션 추가 |

#### 주요 변경사항

1. **Critical Rules 추가**: `Cross-Package Check` 규칙 추가
2. **전문 영역 매트릭스**: 키워드 → 전문 패키지 → 담당 역할 매핑
3. **인계 메시지 포맷**: 표준화된 인계 권유 메시지 정의
4. **예외 사항**: 명시적 요청, 접두사 사용, 공통 기능 등

#### 전문 영역 매트릭스

| 키워드 | 전문 패키지 | 담당 역할 |
|--------|------------|----------|
| Epic, 요구사항, 기획, AC | sax-po | PO |
| 테스트, QA, STG 검증 | sax-qa | QA |
| React, Next.js, 컴포넌트, UI | sax-next | Frontend |
| Spring Boot, Kotlin, API | sax-backend | Backend |
| 배포, Docker, CI/CD, Nginx | sax-infra | DevOps |
| Sprint, 진행도, 할당, 로드맵 | sax-pm | PM |
| 목업, Figma, 디자인, 핸드오프 | sax-design | Designer |
| 마이크로서비스, 이벤트, 워커 | sax-ms | MS Dev |
