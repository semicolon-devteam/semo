# v0.5.0 (2024-11-27)

## Added

### Scripts

- **install-sax.sh 기존 설치 감지 및 처리 기능**
  - 기존 설치 감지 시 인터랙티브 선택 메뉴 제공
    - 1) Update - 최신 버전으로 업데이트
    - 2) Reinstall - 삭제 후 재설치 (클린)
    - 3) Skip - 기존 설치 유지
    - 4) Abort - 설치 취소
  - `--force` 옵션: 기존 설치 강제 삭제 후 재설치
  - `--update` 옵션: 기존 설치를 최신 버전으로 업데이트
  - git index에 등록된 서브모듈 완전 제거 기능 (`remove_submodule`)
  - 색상 출력 강화 (cyan 프롬프트 추가)

## Fixed

- **git index 충돌 문제 해결**
  - 디렉토리가 존재하거나 git index에 등록된 경우 충돌 방지
  - `fatal: already exists in the index` 에러 해결
