# v0.3.0 (2025-12-01)

## ✨ Features

### Commands
- **help.md**: SEMO-QA 패키지 도움말 커맨드 추가
  - QA 워크플로우 가이드 제공
  - 사용 가능한 명령어 목록 표시
  - 단계별 테스트 프로세스 안내

### Skills
- **current-tasks**: 현재 테스트 업무 리스트 조회 스킬 추가
  - GitHub Projects '이슈관리' (ID: 1)에서 '테스트중' 상태 이슈 조회
  - 각 이슈의 QA 테스트 체크리스트 진행 상황 분석
  - 전체 테스트 진행률 계산 및 보고
  - References:
    - `gh-projects.md`: GitHub CLI Projects 명령어 참조
    - `issue-parsing.md`: 이슈 본문 파싱 가이드

## 📝 Notes

이번 릴리스는 QA 테스터의 업무 효율성을 높이기 위한 기능이 추가되었습니다:

1. **help 커맨드**로 언제든지 QA 워크플로우를 확인할 수 있습니다
2. **current-tasks 스킬**로 현재 진행 중인 테스트 업무를 한눈에 파악할 수 있습니다
