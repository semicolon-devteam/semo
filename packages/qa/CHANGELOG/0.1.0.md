# 0.1.0 (2024-11-29)

## Added

### Core Structure

- SEMO-QA 패키지 초기 구조 생성
- CLAUDE.md: QA 테스터용 패키지 설정
- VERSION: 0.1.0
- Orchestrator: QA 워크플로우 라우팅

### Agents

- `qa-master`: QA 전체 프로세스 관리
- `stg-operator`: STG 환경 테스트 실행

### Skills

- `test-queue`: 테스트 대기 이슈 관리
- `validate-test-cases`: AC 기반 테스트 케이스 검증
- `verify-stg-environment`: STG 환경 상태 확인
- `execute-test`: 테스트 실행 및 결과 기록
- `report-test-result`: Pass/Fail 처리 및 상태 변경
- `request-test-cases`: AC 보완 요청
- `iteration-tracker`: 이터레이션 추적
- `production-gate`: 프로덕션 배포 게이트

### Commands

- `/SEMO:test-queue`: 테스트 대기열 확인
- `/SEMO:run-test`: 테스트 실행
- `/SEMO:test-pass`: 테스트 통과 처리
- `/SEMO:test-fail`: 테스트 실패 처리

## Notes
